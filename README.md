# VisuAlis
Create custom UIs for your very own workflows and integrations. Self-hosted, secure, ready OOB.

See `docs` (Note: Documentation may become outdated during development) for the full design. This README is a quickstart.

This project is a sister project of [`AuTomato`](https://github.com/MOBSkuchen/AuTomato), which provides the heavy backend server side logic.
